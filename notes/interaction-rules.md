# Interaction Rules

## Global Principles

整体交互原则：

- 简洁
- 克制
- 高级感
- 不夸张
- 不使用复杂炫技动画

禁止：

- 大幅旋转
- 强弹跳
- 过度缩放
- 科技感过强


--------------------------------------------------

## Navbar

Hover：

- 字体颜色轻微变化
- 底部细线出现

Click：

- 平滑滚动到对应 section


--------------------------------------------------

## Hero Button

Hover：

- 背景亮度轻微提升
- 阴影轻微增强

Click：

- 跳转 Projects 区域


--------------------------------------------------

## About Card

默认状态：

04-about-card-default.png

Hover状态：

04-about-card-hover.png

交互逻辑：

- 鼠标 hover 触发
- 原位置切换
- 不改变布局
- 保持同一尺寸
- 不发生位移


--------------------------------------------------

## Project Card

Hover：

- 卡片轻微上浮
- 图片轻微放大
- 阴影增强

Click：

- 打开项目详情页


--------------------------------------------------

## Footer Social Icons

Hover：

- icon透明度变化
- 微弱位移
