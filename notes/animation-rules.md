# Animation Rules

## Global Animation Principles

整体动画风格：

- 轻量
- 克制
- 顺滑
- 高级
- 慢节奏

禁止：

- 过强科技感
- 复杂3D旋转
- 大范围位移
- 炫技动画


--------------------------------------------------

## Animation Timing

Hover动画：

200ms - 300ms

Section进入动画：

600ms

页面切换：

400ms


--------------------------------------------------

## Animation Easing

使用：

ease-out

保持自然顺滑。


--------------------------------------------------

## About Card Animation

切换方式：

- fade transition
- opacity变化

禁止：

- 翻转
- 弹跳
- 大缩放


--------------------------------------------------

## Project Card Animation

Hover：

- translateY(-4px)
- shadow增强
- 图片scale 1.03

动画节奏：

缓慢、自然。


--------------------------------------------------

## Scroll Animation

Section进入：

- fade up
- opacity 0 → 1
- translateY 20px → 0

禁止：

- 飞入
- 大位移
- 复杂视差


--------------------------------------------------

## Cursor Interaction

鼠标交互：

- 保持轻量
- 不做花哨cursor
