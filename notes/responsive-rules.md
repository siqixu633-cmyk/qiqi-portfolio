# Responsive Rules

## Breakpoints

Desktop：

1920px Design Frame

Content Max Width：

1460px

Tablet：

768px

Mobile：

390px


--------------------------------------------------

## Global Responsive Principles

整体原则：

- 保持信息层级
- 保持视觉节奏
- 避免元素拥挤
- 避免横向滚动

禁止：

- 元素重叠
- 字体过小
- 间距崩坏


--------------------------------------------------

## Navbar

Desktop：

横向排列。

Mobile：

变为汉堡菜单。


--------------------------------------------------

## Hero Section

Desktop：

左右布局。

Mobile：

上下布局。

规则：

- 标题保持居中
- 图片缩小
- 保持视觉焦点


--------------------------------------------------

## About Section

Desktop：

左右布局。

Mobile：

上下布局。

规则：

- 卡片宽度100%
- 保持阅读舒适


--------------------------------------------------

## Project Section

Desktop：

3列布局。

Tablet：

2列布局。

Mobile：

1列布局。


--------------------------------------------------

## Typography

Desktop：

保持大标题视觉冲击。

Mobile：

适当缩小字号。

规则：

- 不压缩行高
- 保持阅读舒适


--------------------------------------------------

## Spacing

Mobile：

减少左右padding。

保持：

16px - 24px安全边距。
